### What I learned

- Vertical alignment of an item.

To make a box vertically center alignment, the following code is useful. 

```html
<body>
  <div class="main">
    ...............
    ...............
  </div>
</body>
```
```css
body{
    position: relative;
    height: 100vh;
}

.main{
    position: absolute;
    top: 50%;
    transform: translateY(-50%);
    width: 100%;
}
```
